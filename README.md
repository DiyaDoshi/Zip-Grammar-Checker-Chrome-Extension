A simple Chrome extension to help you check grammar directly in your browser.

1. **Download the Zip File**  
   Click the green `Code` button and select `Download ZIP` or download `Grammar-Checker-main.zip` directly.

2. **Extract the Zip**  
   Unzip the downloaded file to any folder on your computer.

3. **Open Chrome Extensions Page**  
   - Open Chrome.
   - Go to `chrome://extensions/` in the address bar.

4. **Enable Developer Mode**  
   - Toggle `Developer mode` ON (top-right corner).

5. **Load Unpacked Extension**  
   - Click `Load unpacked`.
   - Select the folder where you extracted the ZIP.

6. **Done!**  
   The extension will be installed and ready to use.
